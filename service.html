個人網頁 Homepage

  • 以帳號為u10116035為例：（自行設定時請換成自己的帳號)
    • 在個人目錄下進入 (或建立) public_html 目錄。 
      mkdir ~/public_html
cd ~/public_html
    • public_html 目錄下的 index.html/index.php 為系統預設之讀取檔案。
    • 網頁位址 URL：在本系首頁後接 ~USERNAME 即可。 
      http://linux1.cs.utaipei.edu.tw/~u10116035/

遠端連線 SSH

  • Windows 使用教學
    • 在 Windows 的 SSH client 可使用 PuTTY、PieTTY 等軟體，連線方式選擇 ssh 後輸入工作站的網域名稱（linux1.cs.utaipei.edu.tw），即可連線至工作站。
  • Mac OSX/Linux 使用教學
    • Linux 與 Mac OSX 都有現成的 Terminal 軟體可以使用，所以只要打開Terminal 輸入： 
      ssh USERNAME@linux1.cs.utaipei.edu.tw
