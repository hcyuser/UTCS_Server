UTCS DNS 計畫

UTCS DNS 是臺北市立大學資訊科學系推動的 DNS 解析服務實驗計畫，提供系上更快速的 DNS 查詢服務。

設定你的裝置

裝置設定

Windows
  • 點選開始列，並選擇進入「設定」項目
  • 選擇進入「網路和網際網路」
  • 選擇進入「變更介面卡選項」
  • 選擇您上網使用的介面設定項目，按右按鍵選擇「內容」項目
  • 選擇「網際網路通訊協定第4版(TCP/IPv4)」項目之後選「內容」進入設定，更改下方的「使用下列的DNS伺服器位址」，更改「慣用DNS伺服器」內容為「10.6.56.151」或者「163.21.235.70」，之後按確定鍵完成設定
  • 您已完成設定! 享受更快及更安全的DNS服務吧!
    • MacOS
  • 請先到「系統偏好設定」下的「網路」設定
  • 選擇您正在連線的連線項目並按下「進階」選項
  • 此時會進入DNS設定畫面請以左下方的「-」號清除原本的DNS設定，之後以「+」號加入「10.6.56.151」或者「163.21.235.70」
  • 之後按「好」,回到上層視窗後按「套用」便完成設定
  • 您已完成設定! 享受更快及更安全的DNS服務吧!
    • iPhone
  • 開啟 「設定」 選項
  • 點選 Wi-Fi項目,然後點選您正連線的WiFi設定檔
  • Tap the - button, and Delete to clean any existing entries.
  • 於列表中按項目前方的「-」號將原本的DNS設定清除
  • 點選「加入伺服器」之後便可以鍵入「10.6.56.151」或者「163.21.235.70」
  • 之後點選右上方「儲存」鍵
  • 您已完成設定! 享受更快及更安全的DNS服務吧!
    • Android
  • 選擇WiFi項目,可見到列出的WiFi設定檔
  • 長按您正在連線的設定檔，選擇「更改網路設定」選項
  • 選擇「顯示進階選項」，之後選擇「DHCP」項目，將其改為「靜態」設定
  • 之後視窗下拉至最後便可見到DNS1 及 DNS2 設定，於DNS1設定處鍵入「10.6.56.151」或者「163.21.235.70」
  • 點選「加入伺服器」之後便可以鍵入「10.6.56.151」或者「163.21.235.70」
  • 之後按下「儲存」
  • 您已完成設定! 享受更快及更安全的DNS服務吧!
